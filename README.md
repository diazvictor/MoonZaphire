# MoonZaphire

MoonZaphire is a GTK3 frontend for MQTT written in Lua with a lot of love.

## Dependencies

- [Lua5.1+](https://www.lua.org/download.html) (or [LuaJIT 2.0+](https://luajit.org/))
- [LGI](https://github.com/pavouk/lgi)


## Running steps

```
lua MoonZaphire.lua
```
